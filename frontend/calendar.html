Aesthenda Calendar

Welcome to the Aesthenda Calendar. Please install and start the full React app to use all calendar features.

To start the full React calendar app:

  1. Open a terminal
  2. Navigate to the client directory: cd client
  3. Install dependencies: npm install
  4. Start the development server: npm start
  5. Access the calendar at: http://localhost:3000
